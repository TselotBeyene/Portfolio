# Tselot Beyene — Portfolio (Tailwind + shadcn-style tokens)

A modern, animated, responsive portfolio website using:
- Tailwind CSS (CDN)
- shadcn-inspired CSS variables/tokens (no build step required)
- Vanilla JS (GitHub repo auto-load + animations)

## Run locally
Just open `index.html` in your browser.

## Deploy
### GitHub Pages
1. Create a repo (e.g. `portfolio`)
2. Upload all files
3. In repo settings → Pages → Deploy from `main` branch `/root`

### Vercel
Create a new project and import the repo (static site deploy).

## Customize
- Edit `index.html` content (headline, about, experience, etc.)
- `script.js` loads your GitHub repos from username: `TselotBeyene`
- Resume file is at: `assets/Tselot_Beyene_Resume.pdf`
